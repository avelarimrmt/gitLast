### Создание коммитов
Для создания коммитов используется команда *git commit*. Для этого в терминале с папкой-репозиторием необходимо написать команду *git commit -m "<сообщение к коммиту>"*. Сообшение к коммиту писать ***ОБЯЗАТЕЛЬНО***.
